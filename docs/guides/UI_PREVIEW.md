### Features Demonstrated

1. **Dynamic Parameter List**
   - Each parameter appears in its own row with an input field
   - Delete button (Ã—) appears on the right side of each row
   - Hover effect highlights the parameter row

2. **Add Button**
   - Prominently displayed below the parameter list
   - Features a plus (+) icon
   - Modern WordPress button styling

3. **Live Counter**
   - Shows the current number of parameters
   - Updates automatically as parameters are added/removed

4. **Visual Feedback**
   - Smooth fade-in animation when adding parameters
   - Smooth fade-out animation when removing parameters
   - Hover effects on input fields and buttons
   - Border highlight on focus
