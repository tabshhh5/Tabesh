# Tabesh Plugin Documentation

This folder contains organized documentation for the Tabesh WordPress plugin.

## Documentation Structure

### `/installation/`
Installation, migration, and quick start guides:
- Installation instructions
- Migration guides for upgrading
- Quick start tutorials

### `/guides/`
User and developer guides:
- Configuration guides (pricing, discounts, settings)
- Feature documentation (file upload, file management, customer panel)
- Visual guides and UI previews
- Persian language guides

### `/api/`
API documentation:
- REST API endpoints
- Integration guides
- API usage examples

### `/security/`
Security-related documentation:
- Security summaries for features
- Security fix documentation
- Best practices

### `/implementation/`
Implementation and fix documentation:
- Feature implementation details
- Bug fix summaries
- Refactoring documentation
- Development notes

### `/troubleshooting/`
Testing and troubleshooting guides:
- Testing guides for features
- Troubleshooting procedures
- Debug information

### `/archived/`
Archived documentation:
- Old implementation summaries
- Deprecated guides
- Historical documentation

## Main Documentation (Root)

The following files remain in the plugin root for easy access:
- `README.md` - Main plugin documentation
- `CHANGELOG.md` - Version history and changes
- `CONTRIBUTING.md` - Contribution guidelines

## Finding Documentation

1. **For installation**: Check `/installation/` folder
2. **For feature usage**: Check `/guides/` folder
3. **For API integration**: Check `/api/` folder
4. **For troubleshooting**: Check `/troubleshooting/` folder
5. **For security info**: Check `/security/` folder
